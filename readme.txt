== Description ==

This widget displays photos from Flickr as a 3D spinning cube. You don't need an API key!

You can show the latest photos from:

*   Someone's public photostream
*   Someone's public favorites
*   A public group pool

= Credits =
The animation is based on the jQuery Image Cube plugin by [Keith Wood](http://keith-wood.name/imageCube.html "jQuery Image Cube").

== Installation ==

1. Unzip the folder.
1. Upload the entire folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the 'Widgets' menu below 'Appearance' to place the widget somewhere in your template
1. WARNING! Some theme's do not support widgets!
